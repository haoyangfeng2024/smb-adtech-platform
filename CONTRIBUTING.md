# Contributing to SMB AdTech Platform

Thank you for your interest in contributing to the SMB AdTech Platform!

## Development Process

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/amazing-feature`)
3. **Commit** your changes (`git commit -m 'Add amazing feature'`)
4. **Push** to the branch (`git push origin feature/amazing-feature`)
5. **Open** a Pull Request

## Code Standards

- Follow [Google Style Guides](https://google.github.io/styleguide/) for each language
- Write meaningful commit messages
- Include tests for new features
- Update documentation accordingly

## Tech Stack

- **Backend**: Python FastAPI, SQLAlchemy, Redis
- **ML Models**: PyTorch (DeepFM, GNN, PPO), scikit-learn
- **Frontend**: React 18, TypeScript, TailwindCSS, Recharts
- **Infrastructure**: Docker, Kubernetes

## Reporting Issues

- Use the provided Issue Templates
- Include detailed reproduction steps
- Attach relevant logs and screenshots
